Design a modern full-featured web application UI called "ZenseExam".

ZenseExam is an automatic multiple-choice exam generator system that allows teachers to manage subjects, courses, question banks, and generate exams both for PDF export and online exams.

Use a modern SaaS dashboard design similar to Notion, Google Classroom, or Stripe dashboard.

------------------------------------------------
DESIGN SYSTEM
------------------------------------------------

Use the following color palette consistently across the system:

Primary Color
#1E3A8A (Deep Blue)
Use for:
- Primary buttons
- Active navigation
- Important UI elements
- Links
- Selected states

Background Color
#F8FAFC (Off-White / Slate)
Use for:
- Main application background
- Page backgrounds
- Dashboard containers

Success / Action Color
#10B981 (Emerald Green)
Use for:
- Success notifications
- Confirm actions
- Exam submitted successfully
- Passed exam status

Warning / Alert Color
#EF4444 (Coral Red)
Use for:
- Error messages
- Failed exam status
- Delete actions
- Important alerts

Body Text Color
#334155 (Dark Slate)
Use for:
- Main text content
- Paragraphs
- Labels
- Form text

Additional UI Guidelines:

- Use white cards with soft shadows for content blocks
- Use rounded corners (8px–12px radius)
- Hover states should slightly darken the primary color
- Use subtle border colors (light gray)
- Use clear visual hierarchy for dashboard content

Typography
- Clean sans-serif font
- Medium weight for headings
- Regular weight for body text

Spacing
- Use generous padding
- Keep layout minimal and readable

Animations
- Smooth hover transitions
- Subtle page transitions

------------------------------------------------
USER ROLES
------------------------------------------------

The system supports 2 roles:

1. Teacher
2. Student

The system must implement Authentication and Authorization with role-based access.

Features:

- Login
- Logout
- Role-based permission
